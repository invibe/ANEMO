dirtree¶

Data.dirtree(dirpath, return_=False)[source]

Allows to display the tree structure of a dirpath data folder.

Parameters

  • dirpath (str) – Data directory path

  • return_ (bool (default False)) – If False prints the path tree in the console, if True does not print the dirpath tree but returns a variable containing it

Returns

tree (str) – Return a variable containing tree of dirpath if return_=True



exemple of code:

[1]:

import ANEMO

#------------------------------------------------------------------------------
# Parameters
#------------------------------------------------------------------------------
dirpath = 'dataANEMO' # Data directory path
return_ = False # If False prints the path tree in the console,
                # if True does not print the dirpath tree but returns a
                # variable containing it

#------------------------------------------------------------------------------
# to apply the function:
#------------------------------------------------------------------------------
ANEMO.Data.dirtree(dirpath, return_)

open¶

Data.open(dirpath, sub=None, task=None, ses=None, acq=None, run=None, RawData=False)[source]

Allows you to open a data folder.

This function returns a python dictionary containing all the data in the directory.

The keys in this dictionary correspond to each particular experiment in the directory, and for each experiment, this dictionary contains a sub-dictionary containing DataFrame pandas:

  • Data - contains the raw data, those calculated from PreProcessing, and the Fit calculated from ProcessingSaccades and ProcessingSmoothPursuit

  • Results - contains the results calculated from ProcessingSaccades and ProcessingSmoothPursuit

  • Events - contains the raw events and those calculated from ProcessingSaccades

  • Settings - contains the settings of the experiment, and function names and their parameters used for Data, Events and Results calculated from ANEMO.

    The Data_column contains the characteristics of the functions used to calculate the data contained in Data and Events, the Results column those used to calculate the data contained in Results.

Parameters

  • dirpath (str) – Data directory path

  • sub (str, or None (default None)) – Participant identifier

  • ses (str, or None (default None)) – Name of the Session

  • task (str, or None (default None)) – Name of the Task

  • acq (str, or None (default None)) – Name of the Aquisition

  • run (str, or None (default None)) – IndexRun

  • RawData (bool (default False)) – If True open RawData, if False open DataAnemo

Returns

data (dict) – Dictionary containing for each experiment the Data, the Results, the Events, and the Settings



exemple of code: to access all the data in the file

[2]:

import ANEMO

#------------------------------------------------------------------------------
# Parameters
#------------------------------------------------------------------------------
dirpath = 'dataANEMO' # Data directory path
sub ='002' # Participant identifier
ses = None # Name of the Session
task = None # Name of the Task
acq = None # Name of the Aquisition
run = None # IndexRun
RawData = False # If True open RawData, if False open DataAnemo

#------------------------------------------------------------------------------
# to apply the function:
#------------------------------------------------------------------------------

# Dictionary containing all the data
full_data = ANEMO.Data.open(dirpath, sub, task, ses, acq, run)
print(full_data.keys())

# Dictionary correspond to particular experiment
#------------------------------------------------------------------------------
data_exp = full_data['sub-002_task-aSPEM']
print(data_exp.keys())

# Access to Data
#------------------------------------------------------------------------------
Data = data_exp['Data']

# Access to Results
#------------------------------------------------------------------------------
Results = data_exp['Results']

# Access to Events
#------------------------------------------------------------------------------
Events = data_exp['Events']

# Access to Settings
#------------------------------------------------------------------------------
Settings = data_exp['Settings']

Data.save(dirpath, data)[source]

Allows you to save a data folder.

Parameters

  • dirpath (str) – Data directory path

  • data (dict) – Dictionary containing for each experiment the Data, the Results, the Events, and the Settings



exemple of code:

ANEMO.Data.save(dirpath, data)

example of code¶

Once your data is open you can modify it.
You will find here two code examples showing you how to do it:

add new data to the Data¶

Here, we want to add a new data in the data that we will call new_data. It corresponds to xpr multiplied by 2, but we want to calculate this new data only for some trials of an experiment

[8]:

import ANEMO

#------------------------------------------------------------------------------
# open Data
#------------------------------------------------------------------------------
# Dictionary containing all the data
full_data = ANEMO.Data.open(dirpath='dataANEMO', sub='002')

expnames = ['sub-002_task-aSPEM'] # list of names of the particular experience
trials = [1,100,110] # list of trials

#------------------------------------------------------------------------------
# add new data
#------------------------------------------------------------------------------
for expname in expnames:

    # Dictionary correspond to particular experiment
    data_exp = full_data[expname]

    # Data to particular experiment
    Data = data_exp['Data'] # or data_exp.Data

    # add new empty data
    Data['new_data'] = [None]*len(Data)

    for t in trials:
        trial_data = Data[Data.trial==t] # trial Data
        Data.loc[(Data.trial==t), 'new_data'] = trial_data['xpr']*2

#------------------------------------------------------------------------------
# save Data
#------------------------------------------------------------------------------
ANEMO.Data.save(dirpath='dataANEMO', data=full_data)


add new events to the Events¶

Here, we want to add a new event in the data that we will call Stop_search_MiSacc. This one corresponds to the event TargetOn + 100ms

[9]:

import ANEMO

#------------------------------------------------------------------------------
# open Data
#------------------------------------------------------------------------------
# Dictionary containing all the data
full_data = ANEMO.Data.open(dirpath='dataANEMO', sub='002')

#------------------------------------------------------------------------------
# add new events
#------------------------------------------------------------------------------
for expname in full_data.keys():

    # Dictionary correspond to particular experiment
    data_exp = full_data[expname]

    # Events to particular experiment
    events = data_exp['Events'] # or data_exp.Events

    # add new events
    events['Stop_recherche_MiSacc'] = events['TargetOn']+100

#------------------------------------------------------------------------------
# save Data
#------------------------------------------------------------------------------
ANEMO.Data.save(dirpath='dataANEMO', data=full_data)
