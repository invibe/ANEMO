ANEMO: a tools for the ANalysis of Eye Movements¶

ANEMO is a python package for preprocessing and processing your eye data.

Before starting, it is important to initialize your data with the init function.

As a first step, it is recommended to pre-process your data using PreProcessing. This will allow you to transform your raw data initially into pixels in degrees or degrees per second, to filter them or to remove or extract your data between two events.

You will be able to process your saccade data in a second time with ProcessingSaccades. In particular by using the detect_misacc function which will allow you to detect saccades and micro-saccades in your data, which were not previously detected by your eye-tracker. You can then remove them from your data or extract them to fit the parameters of a saccade model to your data.

If your data correspond to smooth pursuit eye data, you can use ProcessingSmoothPursuit to extract different parameters. In particular the anticipation velocity, the pursuit latency or the steady state velocity of the pursuit by using classical methods or by fitting parameters of different smooth pursuit models. During this step, it is STRONGLY recommended to use data where saccades and microsaccades have been removed.

You can find all the demo codes in the DEMO section.

Summary¶

  • init - allows to initialize the file which will contain the results of ANEMO

  • Data - sample code to access the results

  • PreProcessing- allows certain calculations to be applied to the data in order to transform it

    • to_deg - transforms position data into degrees

    • to_velocity_deg - transforms position data into degrees per second

    • filter - allows to filter the data

    • remove_events - allows you to remove events from the data

    • extract_events - allows to extract events from data

  • ProcessingSaccades - allows you to apply certain calculations to the data in order to extract parameters

    • detect_misacc - detect micro-saccades

    • remove saccade - sample code to remove saccade and micro-saccades

    • extract saccade - sample code to extract saccade

    • Fit - fits the parameters of a model to the eye data.

      • Saccades fitting function - function to fit the parameters of a model to the eye data.

      • Model.saccade - predefined model reproducing the position of the eye during the saccades

      • GenerateParams.saccade - generate automatically the parameters of the predefined models in Model.saccade in order to fit them to the data.

      • Example of a user-defined model - exemple of fit the parameters of a user-defined model to the eye data.

  • ProcessingSmoothPursuit - allows you to apply certain calculations to the data in order to extract parameters
    • classical method - the “classical method” allowing the extraction of parameters

      • anticipation - the “classical method” allowing the extraction of the velocity of anticipation of pursuit

      • latency - the “classical method” allowing the extraction of the pursuit latency

      • steady_state - the “classical method” allowing the extraction of the steady state velocity

    • Fit - fits the parameters of a model to the eye data.

      • Smooth pursuit fitting function - function to fit the parameters of a model to the eye data.

      • Model.SmoothPursuit - predefined model to fit the data

        • Velocity line - model reproducing the velocity of the eye during the smooth pursuit of a moving target

        • Velocity - model reproducing the velocity of the eye during the smooth pursuit of a moving target

        • Velocity sigmo - model reproducing the velocity of the eye during the smooth pursuit of a moving target

        • Position - model reproducing the position of the eye during the smooth pursuit of a moving target

      • GenerateParams.SmoothPursuit - generate automatically the parameters of the predefined models in Model.SmoothPursuit in order to fit them to the data.

      • Example of a user-defined model - exemple of fit the parameters of a user-defined model to the eye data.

DEMO - code set allowing the extraction of smooth pursuit parameters with the “classical method” and the fit of the velocity_sigmo function to your velocity data starting from your raw data
