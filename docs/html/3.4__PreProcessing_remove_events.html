remove_events¶


Removes the data between two events

Test the parameters¶

PreProcessing.Trial.remove_events(self, xname, trial, expname, Sevents, Eevents, add_stime=0, add_etime=0, start_event=None, stop_event=None, toxname=None, return_=True, **arg)[source]

Removes the xname data between Sevents and Eevents from the trial.

With the parameter return_=True this function returns the calculated data and allows you to test the different parameters.

Parameters

  • xname (str) – Name of the data to be transformed

  • trial (int) – Number of the trial to be transformed

  • expname (str) – Name of the particular experience to be transformed

  • Sevents (str) – Name of the start of the event

  • Eevents (str) – Name of the end of the event

  • add_stime (int (default 0)) – Add time at the start of the event

  • add_etime (int (default 0)) – Add time at the end of the event

  • start_event (str, or None (default None)) – Name of the event marking the start of the search

  • stop_event (str, or None (default None)) – Name of the event marking the end of the search

  • toxname (str (default None)) – Name of the data to be saved

  • return_ (bool (default True)) – If True returns the value, else saves it in data

Returns

data_remove (list) – the calculated data if return_=True



example of code: test the function parameters for removing the raw position x of the eye in pixels between the StimulusOff and TargetOn events.

[1]:

import ANEMO

PreProcessing = ANEMO.PreProcessing(dirpath='dataANEMO', sub='002')

#------------------------------------------------------------------------------
# Parameters:
#------------------------------------------------------------------------------
xname = 'xpr' # The raw position x of the eye in pixels
trial = 8 # Number of the trial to be transformed
expname = 'sub-002_task-aSPEM' # Name of the particular experience to be
                               # transformed
                               # or list(PreProcessing.data.keys())[0]
Sevents = 'StimulusOff' # Name of the start of the event
Eevents = 'TargetOn' # Name of the end of the event
add_stime = 0 # Add time at the start of the event
add_etime = 0 # Add time at the end of the event
stop_event = None # Name of the event marking the end of the search
toxname = None # Name of the data to be saved
return_ = True # If True returns the value, else saves it in data

#------------------------------------------------------------------------------
# to apply the function:
#------------------------------------------------------------------------------
xpr__supp_StimulusOff_TargetOn = PreProcessing.Trial.remove_events(xname,
                                                                   trial,
                                                                   expname,
                                                                   Sevents,
                                                                   Eevents,
                                                                   add_stime,
                                                                   add_etime,
                                                                   stop_event,
                                                                   toxname,
                                                                   return_)

#------------------------------------------------------------------------------
# to retrieve the raw data:
#------------------------------------------------------------------------------
data_exp = PreProcessing.data[expname]
data = data_exp.Data
xpr = data[data.trial==trial][xname].values

#------------------------------------------------------------------------------
# Plot:
#------------------------------------------------------------------------------
import matplotlib.pyplot as plt

fig, axs = plt.subplots(1, 2, figsize=(10, 10/(1.681*2)))

axs[0].plot(xpr, 'k', alpha=.7)
axs[0].set_title('the position x of the eye in pixels: xpr')
axs[0].set_xlabel('time (ms)')
axs[0].set_ylabel('position of the eye (px)')

axs[1].plot(xpr__supp_StimulusOff_TargetOn, 'k', alpha=.7)
new_xname = 'xpr__supp_StimulusOff_TargetOn'
axs[1].set_title('the position x of the eye in pixels:\n%s'%new_xname)
axs[1].set_xlabel('time (ms)')
axs[1].set_ylabel('position of the eye (px)')

plt.tight_layout()
plt.show()

Run it on a data set¶

PreProcessing.Data.remove_events(self, xname, Sevents, Eevents, add_stime=0, add_etime=0, start_event=None, stop_event=None, toxname=None, expnames=None, trials=None, recalculate=False)[source]

Removes the xname data between Sevents and Eevents from the trial and saves it in data.

Allows you to perform this transformation on a data set.

Parameters

  • xname (str) – Name of the data to be transformed

  • Sevents (str) – Name of the start of the event

  • Eevents (str) – Name of the end of the event

  • add_stime (int (default 0)) – Add time at the start of the event

  • add_etime (int (default 0)) – Add time at the end of the event

  • start_event (str, or None (default None)) – Name of the event marking the start of the search

  • stop_event (str, or None (default None)) – Name of the event marking the end of the search

  • toxname (str (default None)) – Name of the data to be saved, if None, toxname will take the value xname+'__supp_'+Sevents+'_'+Eevents

  • expnames (str, or list(str)) – Name or list of names of the particular experience to be transformed, if None all experiences will be transformed

  • trials (int, or list(int), or None (default None)) – Number or list of the trial to be transformed, if None, all the trials will be transformed

  • recalculate (bool (default False)) – Allows you to indicate if you want to force the calculation on the trials already processed



example of code: run this function on the full data for removing the raw position x of the eye in pixels between the StimulusOff and TargetOn events.

[2]:

import ANEMO

PreProcessing = ANEMO.PreProcessing(dirpath='dataANEMO', sub='002')

#------------------------------------------------------------------------------
# Parameters:
#------------------------------------------------------------------------------
xname = 'xpr' # The raw position x of the eye in pixels
Sevents = 'StimulusOff' # Name of the start of the event
Eevents = 'TargetOn' # Name of the end of the event
add_stime = 0 # Add time at the start of the event
add_etime = 0 # Add time at the end of the event
stop_event = None # Name of the event marking the end of the search
toxname= None # Name of the data to be saved,
              # if None toxname will take the value xname+'_deg'
expnames = 'sub-002_task-aSPEM' # Name of the particular experience to be
                                # transformed
                                # or list(PreProcessing.data.keys())[0]
trials = list(range(1, 16)) # List of the trial to be transformed
recalculate = False # Allows you to indicate if you want to force the
                    # calculation on the trials already processed

#------------------------------------------------------------------------------
# to apply the function:
#------------------------------------------------------------------------------
PreProcessing.Data.remove_events(xname,
                                 Sevents,
                                 Eevents,
                                 add_stime,
                                 add_etime,
                                 stop_event,
                                 toxname,
                                 expnames,
                                 trials,
                                 recalculate)
