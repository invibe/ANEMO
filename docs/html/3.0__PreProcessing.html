PreProcessing¶


class ANEMO.PreProcessing(dirpath, sub=None, task=None, ses=None, acq=None, run=None, RawData=False)[source]

PreProcessing allows you to transform your raw data so that you can then perform processing.

This will allow you to transform your raw data into pixels in degrees or degrees per second, to filter them or to remove or extract your data between two events.

  • Use PreProcessing.Trial to test on a trial the different parameters of the functions present in PreProcessing in order to adjust them as well as possible.

  • Once the right parameters are found, you can use PreProcessing.Data to apply the function to a set of data and save it.

Parameters

  • dirpath (str) – Data directory path

  • sub (str, or None (default None)) – Participant identifier

  • task (str, or None (default None)) – Name of the Task

  • ses (str, or None (default None)) – Name of the Session

  • acq (str, or None (default None)) – Name of the Aquisition

  • run (str, or None (default None)) – IndexRun

  • RawData (bool (default False)) – If True open RawData, if False open DataAnemo





Test the parameters

To test the parameters of the different PreProcessing functions, we invite you to use this class:

class PreProcessing.Trial(_)[source]

Allows you to perform PreProcessings on a trial.


example of code:

[1]:

import ANEMO

#------------------------------------------------------------------------------
# Parameters:
#------------------------------------------------------------------------------
dirpath = 'dataANEMO' # Data directory path
sub = '002' # Participant identifier
task = None # Name of the Task
ses = None # Name of the Session
acq = None # Name of the Aquisition
run = None # IndexRun
RawData = False # If True open RawData, if False open DataAnemo
#------------------------------------------------------------------------------

PreProcessing = ANEMO.PreProcessing(dirpath, sub, task, ses, acq, run, RawData)
PreProcessing_Trial = PreProcessing.Trial


Run it on a data set

To apply the PreProcessing functions to a dataset, we invite you to use this class:

class PreProcessing.Data(_)[source]

Allows you to perform PreProcessings on a set of data.


example of code:

[2]:

import ANEMO

#------------------------------------------------------------------------------
# Parameters:
#------------------------------------------------------------------------------
dirpath = 'dataANEMO' # Data directory path
sub = '002' # Participant identifier
task = None # Name of the Task
ses = None # Name of the Session
acq = None # Name of the Aquisition
run = None # IndexRun
RawData = False # If True open RawData, if False open DataAnemo
#------------------------------------------------------------------------------

PreProcessing = ANEMO.PreProcessing(dirpath, sub, task, ses, acq, run, RawData)
PreProcessing_Data = PreProcessing.Data


Functions

This class includes different functions:

  • to_deg - transforms position data into degrees

  • to_velocity_deg - transforms position data into degrees per second

  • filter - allows to filter the data

  • remove_events - allows you to remove events from the data

  • extract_events - allows to extract events from data
